# dancing-mandarins

A dancing drums inspired web slot machine!

In progress. Current features are 
- playing slots with 5 symbols, bets are 8 18 38 68 88
- a multiplier to previously chosen bet amount, x1 x2 x3 x4 x5
- starting with 10000 credits
